# Gymory MVP

## Core goal
Let users find nearby gyms that match their equipment needs — not just by location.

## Target users (Phase 1)
- Strength trainers (powerlifting, bodybuilding)
- HYROX athletes
- Travellers / expats looking for a gym

## MVP screens (v1)
1. **Home** — search bar + quick filter chips (rack, dumbbell, assault bike)
2. **Search results** — list + map toggle
3. **Gym detail** — equipment summary, map, suggest update button
4. **Add gym** — simple form, login required

## MVP filters
- District
- Min rack count
- Min dumbbell weight (kg)
- Has assault bike
- Has ski erg
- Has rower
- Min gym size (sqft)

## Out of scope for v1
- Equipment brand / model details
- User accounts
- Booking / day pass payment
- Social features
- Mobile app
